Ques1. What method is used to serve an HTML file in response to a GET request in Express.js?

Ans: res.sendFile()



Ques2. In a well-structured Node.js application, where should you store HTML files when serving them with res.sendFile()?

Ans: Inside the views/ or public/ folder for better organization.
