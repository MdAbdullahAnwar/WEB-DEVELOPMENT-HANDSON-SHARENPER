Ques1. The property in CSS used to change the background color of an element is

Ans: background-color



Ques2. What does the CSS property display: inline-block; do?

Ans: Display the element as a block-level element with no line break.



Ques3. Which CSS selector has the highest specificity?

Ans: ID selector



Ques4. What does the CSS property box-sizing: border-box; do?

Ans: Includes padding and border in the element's total width and height.



Ques5. What is the purpose of the CSS property z-index?

Ans: Specifies the order of stacked elements along the z-axis.
