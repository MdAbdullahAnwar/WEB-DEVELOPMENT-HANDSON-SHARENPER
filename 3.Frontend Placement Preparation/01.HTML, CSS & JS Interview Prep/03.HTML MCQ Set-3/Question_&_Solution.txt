Ques1. In how many ways can we position an HTML element? What are the permissible values of the position attribute?

Ans: Five ways: static, relative, absolute, fixed, and sticky

Answer Explanation:
Static, relative, absolute, fixed, and sticky are the permissible values of the position attribute in HTML



Ques2. How can you render a website within your own website in HTML?

Ans: Use an <iframe> element with the src attribute set to the URL of the website.

Answer Explanation
Use an <iframe> element with the src attribute set to the URL of the website.

Ques4. In how many ways can you display HTML elements?

Ans: Three ways: Inline, block, and inline-block.
