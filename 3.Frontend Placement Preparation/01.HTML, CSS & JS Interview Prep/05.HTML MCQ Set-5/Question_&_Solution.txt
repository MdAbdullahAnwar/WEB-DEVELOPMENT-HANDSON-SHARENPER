Ques1. In HTML, which attribute is used to specify that an input field must be filled out before submitting the form?

Ans: required



Ques2. What is the purpose of the async attribute in the

Ans: It specifies that the script should be executed asynchronously, allowing it to load in parallel with other elements.



Ques3. What does the readonly attribute do in an HTML input element?

Ans: It specifies that the input field is read-only but can still be submitted with a form.



Ques4. What is the purpose of the defer attribute in the

Ans: It delays the execution of a script until the entire document is parsed.



Ques5. What are Web Workers?
Ans: Web workers allow running JavaScript code in background threads to improve performance.
