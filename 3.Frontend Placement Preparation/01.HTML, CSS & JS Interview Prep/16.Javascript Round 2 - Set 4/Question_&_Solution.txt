Ques1. What is the main difference between the == and === operators in JavaScript?

Ans: == performs type coercion while === does not perform type coercion and checks both value and type.



Ques2. What is the difference between the spread and rest operators in JavaScript?

Ans: The spread operator is used to merge arrays and objects, while the rest operator is used to collect function arguments into an array.



Ques3. Given the code [a, ...b] = [1, 2, 3, 4, 5], what are the values of a and b?

Ans: a is 1, and b is [2, 3, 4, 5].

Ques7. Let arr=[7,8,9,10] Write a function to check if the given number "n" exists in the array. If present remove the number from the array , return the remaining array excluding the number else print element not present. Example Input 1 arr=[7,8,9,10] , n=8 Output arr=[7,9,10] Example Input 2 arr=[7,8,9,10] , n=10 Output "Element not present"

Ans:
function removeElement(arr, n) {
  const index = arr.indexOf(n);
  if (index !== -1) {
    arr.splice(index, 1);
    return arr;
  } else {
    return "Element not present";
  }
}
