Ques1. What is the main difference between the == and === operators in JavaScript?

Ans: == performs type coercion while === does not perform type coercion and checks both value and type.



Ques2. What is the difference between null and undefined in JavaScript?

Ans: null represents an explicitly assigned "empty" or "non-existent" value, while undefined represents an uninitialized or missing value.
