Learn how to serve and use static files like images in a Next.js app.

Watch the video to understand how to add files to the public folder and reference them in your components.



Ques1. Where should you place static assets like images, icons, or fonts in a Next.js project?

Ans: /public



Ques2. How can you reference a file named logo.png located in the /public folder inside a component?

Ans: <img src="/logo.png" />



Ques3. Which of the following statements is true about the /public directory in Next.js?

Ans: Files in /public are served from the root and do not require import statements.



Ques4. If you want to serve a custom favicon.ico in a Next.js app, where should you place it?

Ans: /public/



Ques5. What happens if two files with the same name exist in both /public and /app directories?

Ans: Ony the /public file is accessible via the browser.
